Track growing degree unit accumulation from planting through black layer. Estimates current growth stage, leaf stage, and days to maturity using live weather data.

Soybean GDU Tracker
Monitor soybean development from emergence through R8 maturity. Maps heat unit accumulation to reproductive growth stages using crop-specific base temperatures.

Rainfall Tracker
Compare cumulative in-season rainfall against historical averages for your location. Identify drought stress windows and or excessive rainfall events by growth stage.

About
Data sources
Weather data is pulled live from Open-Meteo's free API using ZIP code coordinates. No API key required. Includes historical actuals and near-term forecast data.
How it works
Enter your planting date and ZIP code. The tool accumulates GDUs from planting forward and maps the total to known growth stage thresholds for each crop.
Who it's for
Built for agronomists, crop advisors, and farmers in the Midwest who want a quick, accurate read on crop development without digging through weather records.
Open source
All tools are MIT-licensed and published on GitHub. Contributions, forks, and feedback are welcome. Built with plain HTML — no build step required.
